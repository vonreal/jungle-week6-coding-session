# SQL 처리기 체크리스트

현재 `member-jiun` 구현 기준으로 요구사항 달성 여부를 정리한 문서입니다.

## 핵심 기능

### CLI
- [x] 텍스트 파일로 작성된 SQL문을 커맨드라인 명령으로 전달 가능
- [x] 실행 형식: `./sql_processor <파일명>`

### SQL 파싱
- [x] INSERT문 파싱 가능
- [x] SELECT문 파싱 가능
- [x] 입력된 SQL 문장을 분석하여 구조화

### 실행
- [x] INSERT 실행 시 데이터를 파일에 저장
- [x] SELECT 실행 시 파일에서 데이터를 읽어서 출력

### 데이터 저장
- [x] 각 테이블은 파일로 관리
- [x] INSERT → 파일에 데이터를 추가
- [x] SELECT → 파일에서 데이터 읽기
- [x] 데이터 저장 포맷 설계 완료
: 현재는 `.data` 기반 커스텀 텍스트 포맷을 사용합니다.

### 전체 조건
- [x] schema 및 table은 이미 존재한다고 가정
- [x] CREATE TABLE은 구현하지 않음
- [x] 구현 언어: C
