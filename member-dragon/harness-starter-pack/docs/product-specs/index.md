# Product Specs Index

이 폴더는 `mini-sql-parser`에서 무엇을 만들어야 하는지 정의한다.

## 기본 문서

- `feature-template.md`: 현재 구현할 SQL 처리기 기능 명세
- `non-functional-requirements.md`: 필요하면 성능/안정성 요구를 추가
- `launch-checklist.md`: 배포나 인수 전 점검이 필요하면 추가
- `localization.md`: 지역별 규칙이 있으면 추가

## 이 프로젝트에서의 역할

- 현재 범위는 `INSERT`/`SELECT`와 에러 처리다.
- 스키마 로딩과 실행 검증의 기준을 적는다.
- 테스트 통과 조건을 문서로 고정한다.
- `member-dragon/plan_codex.md`와 같은 선상에서 읽는다.
